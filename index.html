🌐 GitHub Pages — Gratis, Permanent, Remote Toegankelijk

Je hebt git al geïnstalleerd. Met een gratis GitHub account is dit dashboard binnen 5 minuten overal bereikbaar via een vaste URL zoals https://[jouwNaam].github.io/aanbestedingen. Geen abonnement, geen server nodig.

1

Maak een gratis GitHub account

Ga naar github.com/signup en registreer met een e-mailadres. Volledig gratis.

2

Maak een nieuw repository aan

Na inloggen: klik op "New repository" → naam: aanbestedingen → Public → Create repository. Kopieer daarna de HTTPS clone URL.

3

Configureer git en upload het dashboard

Open een terminal (Win+R → cmd) en voer deze commando's één voor één in (vervang de gegevens door de jouwe):

git config --global user.name "Jouw Naam" git config --global user.email "jouw@email.com" cd C:\Users\raoul\Desktop\aanbestedingen git init git add . git commit -m "Aanbestedingen dashboard" git branch -M main git remote add origin https://github.com/JOUWNAAM/aanbestedingen.git git push -u origin main
4

Activeer GitHub Pages

In je GitHub repository: ga naar Settings → Pages → Source: Deploy from branch → Branch: main → / (root) → Save.

Na ~1 minuut is je dashboard live op:

https://JOUWNAAM.github.io/aanbestedingen/
5

Dashboard bijwerken

Elke keer dat je het dashboard aanpast, voer je dit uit om de wijzigingen online te zetten:

cd C:\Users\raoul\Desktop\aanbestedingen git add . git commit -m "Dashboard update" git push

📱 Notities & Status worden lokaal opgeslagen

De notities en statuswijzigingen die je in het dashboard maakt worden opgeslagen in je browser (localStorage). Ze zijn zichtbaar op de apparaten waar je ze invoert. Wil je notities synchroniseren tussen apparaten? Dan kun je het dashboard aanvullen met een gratis Supabase database — vraag dit aan als je dat wilt.
